## 🏛️ About NayePankh Foundation

[NayePankh Foundation](https://nayepankh.com) is a UP Government, 80G & 12A registered NGO and one of India's largest student-led organisations. Founded during COVID-19, it works toward uplifting underprivileged communities by providing food, sanitary napkins, clothing, and education across cities including Kanpur and Ghaziabad. To date, NayePankh has:

- Helped **2 lakh+ individuals**
- Trained **30,000+ interns**
- Operated across **6+ cities**

---

## 📌 Project Overview

This project fulfils **Option 2** of the Data Analytics Internship task — a fully interactive, single-file web dashboard that analyses NayePankh Foundation's operations across all key verticals.

The dashboard provides data-driven insights across:
- Overall impact and program distribution
- Fundraising trends and donor behaviour
- Volunteer engagement and skill gaps
- Internship batch analytics and geographic reach
- Social media growth and content performance
- Awareness campaign ROI and impact scoring
- Strategic recommendations and a 12-month roadmap

## 💡 Key Insights & Recommendations

The dashboard surfaces six actionable insights:

1. **Beneficiary Tracking** — Introduce Google Forms + Sheets pipeline to record individual aid recipients and measure repeat reach
2. **Donor Churn Prediction** — 62% of first-time donors don't return; a logistic regression model can flag at-risk donors for follow-up
3. **Instagram → Donation Funnel** — 15K followers with weak donation linkage; UTM links + Story stickers can drive 3–5% conversion
4. **Intern-to-Volunteer Pipeline** — Only 6.2% of 30K interns convert; a 30/60/90-day drip campaign can 2× this rate
5. **Seasonal Demand Forecasting** — Oct–Nov shows 40% demand spikes; moving average model enables 6-week advance procurement
6. **Website A/B Testing** — 38% click-to-payment drop-off on donate page; A/B testing CTAs can lift conversion by 15–25%
